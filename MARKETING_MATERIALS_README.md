## 🎨 Design & Branding Guidelines

### Color Palette
```
Primary Blue:   #0066FF (used for buttons, highlights)
Dark Blue:      #001A4D (used for headers, dark mode)
Success Green:  #00CC66 (used for confirmations, checkmarks)
Neutral Gray:   #F5F5F5 (used for backgrounds)
Text Dark:      #1A1A1A (body text)
Text Light:     #808080 (secondary text)
```

### Logo & Wordmark
- Horizontal logo: For headers, long layouts
- Icon/square logo: For avatars, favicons, small spaces
- Wordmark: "TaskerOnChain" always one word, not "Tasker On Chain"

### Typography
```
Headlines: Inter Bold (16-40px)
Body: Inter Regular (14-16px)
Monospace: Courier New (code snippets)
```

### Visual Elements
- Use checkmarks (✅) for completed items
- Use arrows (→) for process flows
- Use icons for features (time icon, robot icon, security icon)
- Use charts for metrics (growth curve, stacked bar, pie)
